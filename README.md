# Jogo RPG de texto em C

Este foi um jogo desenvolvido com a linguagem C, durante o curso de Ciências de Computação no ano de 2019, como trabalho final da matéria de Algoritmos e Programação 1, sendo posteriormente melhorado e aprimorado como trabalho final de Algoritmos e Programação 2.

O projeto foi desenvolvido com o intuito de aplicar os conhecimentos aprendidos nas matérias de AP1 e AP2, lógica de programação e também expandir e trabalhar as habilidades de programação.

Chamado de A Cripta Sombria, este jogo se baseia em RPGs antigos de texto, com um sistema de combate de turnos, um sistema de mochila(com coleta e utilização de itens), sistemas de armadilhas e customização de personagem, no qual você poderá percorrer diversos estágios da cripta e desvendar a história, passando por diversos encontros até encontrar o chefão final, isto é, se você sobreviver.
